チャンネルについて

「ちょもらんまWEBプログラミングChannel」は、株式会社ちょもらんまが運営するYouTube公式チャンネルです。

プログラミングの世界への第一歩を踏み出す方のために、必要なツールの使い方から、実際のコーディング、Webの仕組みまで「見て・触れて・学べる」コンテンツをお届けしています。初心者でも挫折しないよう、分かりやすさを大切にしています。

このチャンネルで学べること

🌐 HTML / CSS

Webサイトのデザインや骨組みを作る基礎から解説。スマートフォン対応のレスポンシブデザインや、Bootstrapを使った効率的なコーディングもカバーします。

🚀 JavaScript

Webサイトに動きをつけるプログラミング言語。基本構文から実践的な動かし方まで、動画を見ながら一緒に手を動かして学べます。

🛠️ ビジネス・開発ツール

Web制作や受講に必要な基本アプリの使い方を解説。多機能ビジネスチャット「Slack」の導入方法や設定など、周辺スキルもバッチリ網羅。

🐱 Scratch (スクラッチ)

直感的にコードを組み立てられるビジュアルプログラミング。まずは「プログラミングが動く楽しさ」を知る第一歩として最適な体験コースを提供します。
